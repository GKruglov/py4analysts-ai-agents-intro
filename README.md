# py4analysts-ai-agents-intro
Вводный курс по Python для системных аналитиков. От парсинга нормативных требований до интеграции с LLM и основ вычислительных графов.

## О курсе

Этот репозиторий содержит материалы вводного практического курса, разработанного специально для системных аналитиков и инженеров. Курс закладывает технический фундамент для изучения интеллектуальных агентов, принципов проектирования графов знаний и концепций нейросимволической интеграции.

Изучение Python строится вокруг решения сквозной утилитарной задачи: создания конвейера (ETL-пайплайна) для автоматизированного парсинга, очистки и классификации требований (на примере стандарта ISO 26262). 

Студенты проходят путь от базовой нормализации строк в функциональной парадигме до написания собственных инструментов (Tools) для ИИ-агентов, взаимодействия с языковыми моделями по REST API и создания базового узла вычислительного графа.

## Структура занятий

Курс состоит из 10 интерактивных Jupyter-ноутбуков, разделенных на логические этапы подготовки данных и моделирования:

### Часть 1. Инженерия данных и функциональный пайплайн
* **Занятие 1. Основы и методы строк:** Извлечение текста и нормализация артефактов форматирования.
* **Занятие 2. Маршрутизация логики:** Условия (`if/else`) и программная отбраковка нерелевантного контекста.
* **Занятие 3. Инкапсуляция:** Написание изолированных чистых функций (базис для инструментов мультиагентных систем).
* **Занятие 4. Итерации:** Циклы (`for`) и пакетная обработка массивов сырых данных.
* **Занятие 5. Списки:** Аккумуляция данных и концепция ограничения контекстного окна (Context Window).
* **Занятие 6. Словари:** Структурирование данных и формирование JSON-объектов с метаданными.
* **Занятие 7. Файловая система:** Чтение выгрузок (ТЗ) и экспорт готовых датасетов в формате JSONL.

### Часть 2. Интеграция с LLM и вычислительные графы
* **Занятие 8. REST API:** Сетевые запросы (`requests`) и интеграция с бесплантыми open-source LLM через платформу OpenRouter для семантической разметки (AI Enrichment).
* **Занятие 9. Введение в ООП:** Классы и магические методы (перегрузка операторов) для инкапсуляции состояний и математических операций.
* **Занятие 10. Основы ML:** Построение базового узла вычислительного графа, аналитическое вычисление производных и реализация шага обратного распространения ошибки (Backpropagation) через рекурсию.

## Требования к окружению
Для прохождения курса потребуется локальная среда разработки:
* IDE: **Visual Studio Code** (с установленным расширением Jupyter)
* Пакетный менеджер: **uv** (рекомендуется для быстрого и надежного управления зависимостями)
